# Revit Bridge — Releases

Public download point for **Revit Bridge** — trigger Autodesk Revit commands from an Elgato
Stream Deck.

## Download & install

1. Download the latest **`RevitBridgeSetup-x.y.z.exe`** from the [**Releases**](../../releases) page.
2. Close Autodesk Revit.
3. Run the installer and follow the prompts. If Windows SmartScreen appears, choose
   **More info → Run anyway**.
4. Reopen Revit.

The installer sets up both halves — the Stream Deck plugin and the Revit add-in — and detects your
Revit version (2025 or 2026) automatically.

## Updates

After installing, add the **Bridge Status** button in the Stream Deck app. It turns amber when a new
version is available; press it to download the latest installer. (The plugin checks `latest.json` in
this repository to know when an update exists.)
